Sprint-3 technical

HTML + CSS

If you were to build a sentence with programming languages, HTML would be the nouns and CSS would be the adjectives. HTML gives you “a cat” but HTML+CSS gives you “a very fluffy orange cat”.

Control Flow

Control flow is the order in which code is read and executed. This is usually from top to bottom but you can control this using javascript as well.

Loops

A loop in JS is something that keeps repeating until the conditions are met. It’s like when you brush your hair. You repeat the motions until all the knots are out and then you stop. You don’t just keep brushing indefinitely and the number of brushes can vary depending on how many knots you had in the first place.

The DOM

The DOM, or Document Object Model is what we use to interact with web pages. We can use the DOM to add, delete or change content on an html document.

Arrays

An array is a special variable, which can hold more than one value at a time. It acts as a list of what is inside the variable. You can access the values by referring to an index number. Instead of

var cat1 = "Breezy";

var cat2 = "Pickles";

var cat3 = "Frodo";

You can now have var cats = ["Breezy", "Pickles", "Frodo"];

To access data from an array, you can refer to an index number. For example var name = cats[0]; will access breezy because she is the first element and array indexes start with [0] instead of [1]

Objects

An object is a container for named values called properties or methods. Just like a real object you can give it properties to describe it. For example:

var cat = {firstName:"Breezy", lastName:"Meow", age:18, eyeColor:"Brown"};

You can access properties from an object in two different ways. Either with a dot or with square brackets and quotes. This is seen as breezy.lastName; or breezy["lastName"];

Functions

A function is a a set of statements that performs a task or calculates a value. The code inside the function will execute when something calls the function. This can be a few things like when an event occurs (when a user clicks a button) or when it is invoked (called) from JavaScript code.

Functions are useful because you can outline a set of instructions that you can apply lots of different times, to different things so you don’t have to write out the same “instructions” over and over again.
