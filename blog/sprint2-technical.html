What is the difference between Margin, Border, and Padding?

25th April 2019

To understand margins, borders, and padding we have to look at something called the CSS Box Model. You can see it below.

It is essentially a box that wraps around every single html element (an element is any content that appears on the page - it could be a heading, a paragraph, an image… the list goes on). The box model contains different parts.

  • Content - the filling of the box, where text and images and all the fun stuff apears.
  • Padding - Clears an area around the content. The padding is transparent- you don’t see it’s there but you can definitely notice when it’s not.
  • Border - A border that goes around the padding and content. This can be fun colours and patterns like
    this
  • Margin - The margin clears an area outside the border. The margin is also transparent. Almost invisible BUT important

All of these elements together allow us to add a border around elements, and to define space between elements.
